# VirtualPiano

## Instalacja

* Klonujemy projekt
* Do działania potrzebujemy zainstalowanego środowiska `node.js` oraz `yarn`.
* Przechodzimy do katalogu z projektem i wykonujemy komendy:

```
yarn install
yarn start
```

* Projekt powinien uruchomić się w przeglądarce pod adresem `localhost:3000`
